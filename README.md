# 🦁 콘솔 기반 아기 사자 명단 관리 프로그램

자바의 핵심 문법인 데이터 타입, 입출력, 조건문, 반복문을 활용하여 제작한 콘솔 기반 명단 관리 프로그램입니다. 프로그램의 가장 기본적인 흐름인 **입력 → 처리 → 출력** 과정을 직접 구현하며 자바의 기초를 다지는 것을 목표로 합니다.

## 🎯 과제 목표

* **자바 기초 문법 습득**: 데이터 타입, 변수, 연산자의 개념 이해 및 활용
* **콘솔 입출력 구현**: `Scanner`를 통한 사용자 입력 처리와 콘솔 출력 흐름 이해
* **제어문 활용**: 조건문과 반복문을 사용하여 프로그램의 실행 흐름 제어
* **데이터 저장 및 관리**: 배열(Array)을 활용하여 여러 데이터를 효율적으로 저장하는 방법 습득

## 🛠 기술 스택

* **Language**: Java (JDK 17 이상 권장)
* **Environment**: Console (CLI)

## 📋 주요 기능

1. **인원수 설정**: 관리할 아기 사자의 총 인원수를 사용자로부터 입력받습니다.
2. **명단 입력**: 설정된 인원수만큼 아기 사자의 이름을 차례대로 입력받아 배열에 저장합니다.
3. **최종 명단 출력**: 저장된 모든 이름을 확인하기 쉬운 형태로 콘솔에 출력합니다.

## 🚀 실행 흐름 (Implementation Logic)

1. **Input**: `Scanner` 클래스를 사용하여 아기 사자의 수를 정수형으로 입력받습니다.
2. **Process**: 입력받은 수의 크기만큼 `String` 배열을 생성하고, 반복문을 통해 각 칸에 이름을 저장합니다.
3. **Output**: 저장된 배열을 순회하며 입력된 전체 명단을 화면에 보여줍니다.

## ⚠️ 제약 사항 및 유의 사항

* 자바 표준 라이브러리(Java Standard API)만을 사용하여 구현합니다.
* 예외 상황(예: 숫자가 아닌 값 입력 등)에 대한 처리를 고려하여 작성하면 더욱 좋습니다.
* 명확한 변수명과 주석을 통해 코드의 가독성을 높입니다.
